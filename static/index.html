    Manovra Prediction Platform Guide

    Tutorial

    Welcome to the Manovra Behavior Prediction Console

    This dashboard is a live playground demonstrating how recurrent neural networks (RNNs) forecast sequences of user actions in real-time across different domains. By observing behavioral patterns, the system calculates the probability distribution of subsequent actions.

    1. Select a Domain Preset

    Toggle between SaaS App, Gaming, Streaming, and E-Commerce. The actions and users will dynamically adapt.

    2. Ingest Actions

    Click preset buttons (like view_dashboard, create_project) or type custom events. Every action updates the user history sequence.

    3. View Predictions

    Watch the AI Prediction Dashboard. The LSTM updates and displays the top 3 predicted next actions with percentage bars.

    4. Fit the Model

    Go to the Telemetry & Training tab. Click "Batch Simulate" to load data, then click "Fit LSTM Model" to train the neural network live.

    How the Model Works Under the Hood

    Recurrent Sequence Processing (LSTM): Unlike standard classifiers, the neural network uses a Long Short-Term Memory (LSTM) cell. When a sequence is processed, each event type is hashed into an index and mapped to a dense embedding. The LSTM steps through this embedding sequence, updating its internal hidden state. The final state vectors represent the entire context of user behavior, which are passed to a Linear layer predicting the next activity.
